Get Running in Minutes

Clone, configure, and launch. RedAgent runs on any system with Python 3.11+.

01 · CLONE & INSTALL
# Clone the repository
git clone https://github.com/Insider77Circle/Red-Agent.git
cd Red-Agent

# Install dependencies
pip install -r requirements.txt
02 · CONFIGURE API KEY
# Copy the environment template
cp .env.example .env

# Add your DeepSeek API key
# Get one free at platform.deepseek.com
nano .env

# Set this value:
DEEPSEEK_API_KEY=sk-your-key-here
03 · LAUNCH & OPERATE
# Launch RedAgent
python main.py

# Example commands to try:
"Find me fast proxies and check DNS leaks"
"Build a 3-hop chain through Europe"
"Run nmap through my proxy chain"
